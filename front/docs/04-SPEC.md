# 스펙 (Spec)

> 도메인 로직과 구현할 기능을 정의한다. 무엇을 만들 것인지 명확히 한다.

---

## 프로젝트 개요

**video-textable**: 마크다운을 입력하면 음성(TTS) + 영상으로 변환하는 브라우저 기반 서비스

- 백엔드 없음. 모든 처리는 브라우저에서 실행된다.
- 음성은 브라우저 내장 Web Speech API(`SpeechSynthesis`)를 사용한다.
- 롤 모델: [markdown_video](https://github.com/Avik-creator/markdown_video)

---

## 핵심 흐름

```
마크다운 입력 → 씬(Scene) 파싱 → 미리보기 렌더링 → TTS 음성 생성 → 영상 녹화/인코딩 → 다운로드
```

---

## 기능 목록

### 1. 마크다운 에디터 — ✅ 100%

- [x] 마크다운 텍스트를 입력할 수 있는 에디터를 제공한다.
- [x] 글자 수를 실시간으로 표시한다.
- [x] 씬 구분자(`---`)로 여러 씬을 나눌 수 있다.

### 2. 씬(Scene) 파싱 — ✅ 100%

- [x] 마크다운을 `---` 기준으로 씬 단위로 분리한다.
- [x] 각 씬의 마크다운을 HTML로 변환한다.
- [x] 씬 목록을 배열로 관리한다.

### 3. 미리보기 — ✅ 100%

- [x] 파싱된 씬을 슬라이드 형태로 미리보기한다.
- [x] 이전/다음 씬 이동이 가능하다.
- [x] 현재 씬 번호를 표시한다.
- [x] 마크다운 변경 시 미리보기가 실시간 반영된다.

### 4. TTS (Text-to-Speech) — 🔧 60%

- [x] 브라우저 내장 `SpeechSynthesis` API를 사용한다.
- [x] 각 씬의 텍스트를 음성으로 읽는다.
- [x] 한국어 음성을 기본으로 사용한다.
- [ ] 음성 속도(rate), 높이(pitch) 조절 UI를 제공한다.
- [x] 재생/일시정지/정지 컨트롤을 제공한다.

### 5. 재생 (Play) — 🔧 75%

- [x] 전체 씬을 순서대로 재생한다.
- [x] 씬 전환 시 TTS가 해당 씬의 텍스트를 읽는다.
- [x] TTS 읽기가 끝나면 자동으로 다음 씬으로 넘어간다.
- [ ] 재생 중 현재 진행 상태를 표시한다. (프로그레스 바 등)

### 6. 씬 이미지 내보내기 — ✅ 100%

- [x] 현재 보고 있는 씬을 PNG 이미지로 캡처한다. (`html2canvas`)
- [x] 캡처한 이미지를 다운로드할 수 있다.
- [x] 내보내기 버튼을 제공한다.

### 7. 영상 녹화 및 다운로드 — ⬜ 0%

- [ ] 미리보기 영역을 캡처하여 영상으로 녹화한다. (`html2canvas` + `FFmpeg.wasm`)
- [ ] 녹화된 영상을 다운로드할 수 있다.

## 기술 스택

| 영역 | 기술 |
|---|---|
| 프레임워크 | Next.js 16 (App Router) |
| 언어 | TypeScript |
| 스타일 | Tailwind CSS 4 + shadcn/ui |
| 마크다운 파싱 | 자체 구현 (정규식 기반, 추후 라이브러리 교체 가능) |
| 음성 | Web Speech API (`SpeechSynthesis`) |
| 이미지 캡처 | html2canvas-pro |
| 영상 녹화 | 향후: html2canvas + FFmpeg.wasm |
